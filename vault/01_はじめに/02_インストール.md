macOSまたはWindowsでは、公式サイトからHermes Desktopインストーラーを取得する方法が推奨されています。すでにHermes AgentのCLIを導入済みの場合は、ターミナルから`hermes desktop`を実行してDesktop Appを起動できます。

![](@editorial/editorial-12-hermes-sandals.webp)

## macOS・Windows

1. [公式Downloadページ](https://hermes-agent.nousresearch.com/)を開く。
2. OSに合うHermes Desktopインストーラーを取得する。
3. ダウンロードしたインストーラーを実行する。
4. Hermes Desktopを起動し、初回セットアップ画面が表示されることを確認する。

配布物のファイル名、署名、更新方法は変更される可能性があります。第三者のミラーではなく公式サイトから取得してください。

## CLI導入済みの場合

```bash
hermes desktop
```

この方法では、現在の設定、キー、セッション、スキルをDesktop Appと共有します。

## Linux

公式ガイドではDesktop AppはLinuxでも動作するとされています。CLIを公式インストーラーで導入し、`hermes desktop`を実行する方法を確認してください。

```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
hermes desktop
```

:::caution[シェルスクリプトを実行する前に]
組織のセキュリティ方針がある場合は、スクリプトの内容と配布元を確認してから実行してください。管理対象端末では管理者の手順を優先します。
:::

## 起動できたか確認する

- Hermes Desktopのウィンドウが開く
- 初回セットアップまたはチャット画面が表示される
- 設定画面を開ける

起動に失敗する場合は[[05_トラブルシューティング/01_問題の切り分け|問題の切り分け]]を確認してください。
