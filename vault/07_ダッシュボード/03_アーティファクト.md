> v0.17 実機（ビルド `857d024`）のアーティファクトパネルで確認。件数（すべて / 画像 / ファイル / リンク）は環境依存の例。

左ナビの「アーティファクト」では、セッションを横断して生成・参照した成果物（画像・ファイル・リンク）を一覧できる。どのチャットで作ったか分からなくなった画像や、過去に参照した URL を素早く探すためのパネル。

## タブ（種別フィルタ）

上部のタブで種別を絞り込む。

| タブ | 内容 |
|---|---|
| すべて | 画像・ファイル・リンクをまとめて表示 |
| 画像 | 生成・添付した画像 |
| ファイル | 読み書きしたファイル（ローカルパスなど） |
| リンク | 参照した URL |

右上の検索でアーティファクトを絞り込める。件数が多いタブはページ送りで表示する。

## 表示形式

アーティファクトは種別によって表示が分かれる。

- **画像**: サムネイル付きのカードで並ぶ。
- **ファイル・リンク**: 表形式の行で並ぶ（サムネイルなし）。

いずれも次の情報を持つ。

- 種別（画像 / ファイル / リンク）
- ファイル名 / URL
- 保存先パスまたは取得元 URL（例: `~/Downloads/...`、`/tmp/...`、`https://...`）
- 生成元のセッション（どのチャットで作られたか）
- 生成元チャットへのジャンプ（画像はカードの「チャット」ボタン、ファイル・リンクは行の「セッション」セル）

## 前提・操作・確認

- 前提: これまでのセッションで画像・ファイル・リンクを生成 / 参照していること。
- 操作: タブで種別を絞るか検索し、目的のアーティファクトを開く。生成元チャットへのジャンプで元の会話に戻れる。
- 期待結果: 過去の成果物を横断的に探し、元のセッションをたどれる。
- 見つからないとき: タブの種別、検索語、ページ送りを確認する。
