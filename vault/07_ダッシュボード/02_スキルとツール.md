> [!note] 確認状況
> v0.17 実機（ビルド `857d024`）のスキルとツールのパネルで確認。スキル数・カテゴリ・有効数は環境により変わる。

左ナビの「スキルとツール」では、エージェントが使えるスキルとツールセットを一覧し、トグルで有効 / 無効を切り替える。上部の「スキル」「ツールセット」タブで切り替え、右上の検索で絞り込める。

## スキル

スキルはカテゴリ別に並ぶ。各スキルに説明とトグルがある。実機例では 103 スキルが次のカテゴリに分かれていた。

- **Apple**（apple-notes、apple-reminders、findmy、imessage、macos-computer-use ほか）
- **Autonomous-AI-Agents**（claude-code、codex、hermes-agent、opencode ほか）
- Creative、Data-Science、Devops、Email、Gaming、General、Github、Mcp、Media、Mlops、Note-Taking、Productivity、Red-Teaming、Research、Smart-Home、Social-Media、Software-Development

> [!tip] スキルとは
> スキルは特定の作業手順やツール群をまとめた拡張。有効にしたスキルだけがエージェントから使える。

## ツールセット

ツールセットは、関連するツールをまとめた単位。各ツールセットには含まれるツール（タグ）と状態が表示される。実機例では 26 ツールセット中 19 が有効だった。

| ツールセット | 含まれるツール（例） | 状態 |
|---|---|---|
| Browser Automation | `browser_navigate`、`browser_click`、`browser_type`、`web_search` ほか | 設定済み |
| Clarifying Questions | `clarify` | 設定済み |
| Code Execution | `execute_code` | 設定済み |
| Computer Use (macOS) | `computer_use`（cua-driver 経由のデスクトップ操作） | 設定済み |
| Context Engine | アクティブなコンテキストエンジンのツール | オフ（既定で無効） |
| Cron Jobs | `cronjob`（作成・一覧・更新・停止・再開・実行） | 設定済み |
| Discord (read/participate) | `discord` | 設定済み |
| Discord Server Admin | `discord_admin` | 設定済み |
| File Operations | `read_file`、`write_file`、`patch`、`search_files` | 設定済み |
| Home Assistant | `ha_call_service`、`ha_get_state`、`ha_list_entities`、`ha_list_services` | キーが必要 |
| Image Generation | `image_generate`（生成・編集） | 設定済み |

「キーが必要」は、API キー等の設定が済むまで有効化できないツールセット。

## 前提・操作・確認

- 前提: スキル / ツールセットの一覧が表示できること。
- 操作: 使いたいスキル・ツールセットのトグルを ON にする。「キーが必要」のものは先にキーを設定する。
- 期待結果: 有効にしたスキル・ツールがエージェントから使える。
- 使えないとき: トグルが ON か、「キーが必要」表示が残っていないか、対象の API キー設定を確認する（[[03_設定/11_ツールとキー|ツールとキー]]・[[03_設定/12_MCP|MCP]]）。
