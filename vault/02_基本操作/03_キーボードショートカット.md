> [!note] 確認状況
> v0.17 実機（ビルド `857d024`）のキーボードショートカットパネルで確認。既定キーは下表のとおりだが、各自リバインドできるため表示と異なる場合がある。キー表記は macOS のもの。

`⌘/`（Command + スラッシュ）でキーボードショートカットのパネルを開く。各行をクリックすると、その操作にキーを割り当て直せる（リバインド）。右上の「Reset all」で既定に戻す。

> [!tip] パネルの開き方
> `⌘/` でいつでもこのパネルを再表示できる。

## コンポーザー（COMPOSER）

| 操作 | 既定キー | 説明 |
|---|---|---|
| Focus composer | 未設定 | 入力欄にフォーカス |
| Open model picker | 未設定 | モデル選択を開く |
| Send message | `↵` | メッセージを送信 |
| Insert newline | `⇧ ↵` | 改行を挿入 |
| Steer the running turn | `⌘ ↵` | 実行中のターンに割り込んで方向づけ |
| Send next queued turn | `⌘ ⇧ K` | 次のキュー済みターンを送信 |
| Reference files, folders, URLs | `@` | ファイル・フォルダ・URL を参照 |
| Slash command palette | `/` | スラッシュコマンドパレットを開く |
| Quick help | `?` | クイックヘルプ |
| Cycle popover / history | `↑` `↓` | ポップオーバー / 履歴を循環 |
| Close popover · cancel run | `Esc` | ポップオーバーを閉じる・実行を取り消す |

「未設定」の操作は既定キーが割り当てられておらず、パネルでリバインドすると使えるようになる。

## プロファイル（PROFILES）

| 操作 | 既定キー |
|---|---|
| Switch to default profile | `⌘ D` |
| Switch to profile 1〜9 | `⌘ 1` 〜 `⌘ 9` |
| Switch to profile 10 | `⌘ ⌥ 1` |
| Switch to profile 11 | `⌘ ⌥ 2` |

切り替え先のプロファイルは [[07_ダッシュボード/01_プロファイル管理|プロファイル管理]] で編集する。

## 前提・操作・確認

- 前提: Desktop アプリがフォアグラウンドにあること。
- 操作: `⌘/` でパネルを開き、変更したい操作をクリックして新しいキーを押す。競合する場合は再割り当てを促される。
- 期待結果: 設定したキーで操作が実行される。`Reset all` で既定に戻る。
- うまく動かないとき: 他アプリや OS のショートカットと競合していないか、リバインドで上書きしていないかを確認する。
