## 失敗時の撤退

- frontmatter 降格で preprocess が落ちる → 落ちたファイルの YAML を確認（`status` の値・インデント）。`src/content.config.ts` の enum（`verified`/`needs-review`）と一致させる。
- 取り返しが必要な場合: `git restore vault/` で frontmatter 変更を破棄（ブランチは残す）。ブランチごと捨てるなら `git switch main && git branch -D docs/hermes-v0.17`。
- 人間への報告事項: 実機バージョンが 0.17 でなかった/情報画面が想定と違った場合は、採取値をそのまま報告して指示を仰ぐ。
